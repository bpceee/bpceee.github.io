11:Tf5a,<html><head></head><body><p dir="auto">It's about time to start my own tech blog!</p>
<p dir="auto">At first glance, <a href="https://github.com/jekyll/jekyll">Jekyll</a> is quite a good choice, but it's a little bit heavy in some aspects (it uses Ruby and needs to commit every time when post). As a minimalist, I decided to utilize <a href="https://developer.github.com/v4">Github GraphQL API v4</a> as data source and <a href="https://pages.github.com">Github Pages</a> to host frontend code. More specifically, this blog use a repo's issues as blog posts and a simple React Web App to load and present these issues. It's totally free, highly configurable and has a  in-place comments board.</p>
